Introduction

jQuery is a JavaScript library.

It is designed to:

  • facilitate navigating a document
  • selecting DOM elements
  • event handling
  • creating animations
  • developing Ajax applications, etc.

How to Use

jQuery can be downloaded from jQuery.com.

You should reference it with a <script> tag inside the <head> section of your document. 

          
 <head>
   <script src="jquery-3.4.0.min.js"></script>
 </head>
Syntax

jQuery syntax is tailored for selecting (querying) HTML elements and performing some action on them.

The basic syntax is: $(selector).action()

Here is an example:

          
 <div id="hello">
   <p>Hi there!</p>
 </div>
        
 $(function() {
   $("#hello").text("Hello World!");
 });
Events

One common example of a jQuery event method is $(document).ready()

It is used to execute a function when the document is fully loaded.

Here are some more useful examples of event handler methods:

Hiding an element fired by a click event:

        
 $("p").click(function(){
   $(this).hide();
 });

This method attaches an event handler function to a form field. The function is executed when the field loses focus:

        
 $("input").blur(function(){
   $(this).css("background-color", "#ffffff");
 });
Get and Set

There are three useful jQuery methods for DOM manipulation: text(), html(), and val().

Another method, attr(), is used to get attribute values. It also allows you to set multiple attributes at the same time.

For more information about these methods, please refer to the official documentation.
